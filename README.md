**PlanificatorOS** es un entorno de tablero de control (dashboard) y gestión de tiempo ultraligero que opera directamente en el navegador. Un diseño estetica dark modern, y su facil uso intuitivo.

---

## Características Principales

- **Motor de Tiempo Real (Kernel):** Reloj de precisión y calendario dinámico que se actualizan sin bloqueos del hilo principal.
- **Protocolo de Alarma Integrado:** Sistema de notificaciones basado en tiempo con validación estricta a nivel de segundos y feedback visual inmediato.
- **Arquitectura Zero-Dependencies:** Construido puramente en Vanilla HTML5, CSS3 y JavaScript moderno. No requiere frameworks (React, Vue, etc.) ni procesos de compilación (Webpack, Vite).
- **Capa de Seguridad Frontend:** Prevención integrada contra la apertura accidental de herramientas de desarrollo (F12, Ctrl+Shift+I) para mantener la inmersión del entorno (usando mi codigo block_keys).
- **Totalmente Responsivo:** El diseño de cuadrícula (Grid/Flexbox) se adapta perfectamente a pantallas de escritorio, tablets y dispositivos móviles.
